Polynomial Addition

Two file names, each representing a polynomial, are given in the command line. The program expects a polynomial such as 9*x^4 + 8*x^3 + 7*x^2 + 6*x^1 + 5 to be represented as 5 9 8 7 6 5 where the first number is the length of the polynomial and the following numbers are the coefficients of the polynomials in order. The program expects the highest degree coefficient to be non-zero.

How it Works

This program checks which input polynomial has the greater length (higher degree). This polynomial's length is the length of the resulting polynomial. Its coefficient are copied into the resulting polynomial until an exponent that both polynomials contain is reached, at which point the coefficients will be added.

Security Considerations

This program was designed from the start with the the Memory Management (MEM) and Characters and Strings (STR) rules and recommendations from the SEI CERT C Coding Standard and CERT C++ Coding Standard. Additionally, teams reviewed each other's code after the development stage ended for strengths and weaknesses based on these rules.
